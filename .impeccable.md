## Design Context

### Users
**主要用户**：求职者
- 场景：需要快速制作针对特定 JD 的专业简历
- 目标：高效生成与 JD 高度匹配的简历，提升面试通过率
- 使用情境：可能在找工作期间频繁使用，需要快速迭代多个版本

### Brand Personality
**3 个关键词**：专业、可信、高效

- 专业：体现职业素养，让招聘方信任
- 可信：界面稳重，功能可靠，数据安全
- 高效：操作流畅，响应快速，不浪费用户时间

### Aesthetic Direction
**视觉风格**：现代设计感

- 清晰的信息层级和视觉层次
- 现代化的 UI 元素（圆角、阴影、渐变）
- 合理的留白和呼吸感
- 精致但不花哨的微交互

**主题模式**：保留并优化现有的三主题系统
- Default (Claude)：专业简约，默认工作主题
- Retro-Mac (🖥️)：复古像素风格，个性化体验
- Sanrio-Claw (🎀)：可爱梦幻风格，个性化体验

### Design Principles
1. **专业优先**：所有设计决策以体现专业性为第一原则
2. **效率至上**：减少点击次数，操作直观，反馈即时
3. **清晰层级**：信息架构清晰，视觉层次分明
4. **克制动效**：动画服务于功能，不喧宾夺主
5. **一致性**：三主题系统保持体验一致，风格差异可控
