# Malleable Resume - 前端

AI 驱动的可塑排版简历系统。

## 功能特性

- 📄 AI 简历解析（支持 PDF/Word/图片）
- 🎯 JD 驱动的简历改写
- 🔄 弹簧拖拽调节文本长度
- 💬 AI Copilot 对话
- 🎨 实时排版工作台

## 部署

本项目前端托管于 GitHub Pages。

## 后端 API

需要配合后端服务器使用，后端地址：
- 开发环境：`http://localhost:3000`
- 生产环境：待部署

## 本地开发

```bash
# 启动前端
npx http-server -p 8080

# 启动后端（另开终端）
cd server && node index.js
```

访问：http://localhost:8080/test3.html
